# Swiggy-Sales-Dashboard-Analysis
This project presents an Excel-based interactive dashboard analyzing Swiggy food order data from January 2025 to August 2025. The dashboard provides insights into order trends, restaurant performance, food categories, and pricing patterns across multiple cities in India. 
The goal of this project is to convert raw food delivery data into visual insights using Excel dashboards and pivot tables.
Dataset Information

The dataset contains 197,430 orders with the following attributes:-
State
City
Order Date
Day / Week / Quarter
Restaurant Name
Location
Category
Dish Name
Food Type (Veg / Non-Veg)
Price (INR)
Rating
Rating Count

Key Insights from the Data :-
Total Orders
Average Food Price
Top Cities by Orders
Food Type Distribution

Dashboard Features :-
Monthly sales trend analysis
City-wise order distribution
Food category performance
Veg vs Non-Veg comparison
Restaurant ratings analysis
Interactive slicers and filters
